Database notes

# Relational style

## User table
  [pk] userid
  username (email address)
  datestamp

## Polls table
  [pk] pollid 
  [fk] userid 
  poll title
  datestamp 

## Votes table
  [pk] voteid
  [fk] pollid
  [fk] userid
  vote detail
  datestamp
